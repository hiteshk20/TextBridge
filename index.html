TextBridge

Send text to any device. No app install. No login. No cost.

  1. Open this website on the device that has the text.
  2. Paste the text and create a secure link.
  3. Send the link to the other person through WhatsApp, email, SMS, etc.
  4. They open the link in their browser and read the text instantly.

If you leave this blank, the password is included in the link so the other person only needs to open it. If you set a password, share it with them separately.
